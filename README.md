# QuadArt

Simple module for generating QuadTree art

## Example usage

### Through Shell

```bash
wget https://image.freepik.com/free-photo/green-apple-with-leaves_1101-453.jpg
./quadart.py green-apple-with-leaves_1101-453.jpg --thresh 40
```
